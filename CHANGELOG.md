# История изменений

## Версия 1.1.0 (2025-11-13)

### Добавлено
- **Генерация случайных ID вместо номеров телефона**
  - Возможность создавать ключи без номера телефона
  - Автоматическая генерация ID формата `user_XXXXXXXXXX`
  - Поддержка произвольных идентификаторов (текст, номера, и т.д.)

- **QR коды для ключей**
  - Автоматическая генерация QR кода для каждого созданного ключа
  - Отправка QR кода в виде изображения в чат
  - Возможность сканирования QR кода в VPN приложениях

### Изменено
- Кнопка "Сгенерировать номер" изменена на "Сгенерировать ID"
- Улучшено форматирование сообщений при создании ключа
- Добавлены эмодзи для лучшей читаемости
- Обновлена валидация ввода (минимум 3 символа для ID)

### Технические изменения
- Добавлены библиотеки: `qrcode[pil]==7.4.2`, `pillow==10.1.0`
- Добавлен модуль `qr_generator.py` для генерации QR кодов
- Обновлена функция `generate_user_id()` в `validators.py`
- Улучшена обработка ошибок при генерации QR кодов

## Версия 1.0.0 (2025-11-13)

### Добавлено
- Первый релиз бота
- Система ролей (Администратор/Менеджеры)
- Создание VLESS ключей через X-UI API
- Выбор срока действия ключа (1, 3, 6, 12 месяцев)
- Автоматическая установка лимита 2 IP на ключ
- База данных SQLite для хранения менеджеров и истории
- Статистика для менеджеров и администратора
- Панель администратора для управления менеджеров
